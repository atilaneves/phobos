    Module std.stream

    Warning: This module is considered out-dated and not up to Phobos' current standards. It will remain until we have a suitable replacement, but be aware that it will not remain long term.

    Interfaces

    Name Description
    InputStream InputStream is the interface for readable streams.
    OutputStream Interface for writable streams.

    Classes

    Name Description
    BufferedFile This subclass is for buffered file system streams.
    BufferedStream This subclass is for buffering a source stream.
    EndianStream This subclass wraps a stream with big-endian or little-endian byte order swapping.
    File This subclass is for unbuffered file system streams.
    FilterStream A base class for streams that wrap a source stream with additional functionality.
    MemoryStream This subclass reads and constructs an array of bytes in memory.
    MmFileStream This subclass wraps a memory-mapped file with the stream API. See std.mmfile module.
    OpenException An exception for errors during File.open.
    ReadException Thrown when unable to read data from Stream.
    SeekException Thrown when unable to move Stream pointer.
    SliceStream This subclass slices off a portion of another stream, making seeking relative to the boundaries of the slice.
    Stream Stream is the base abstract class from which the other stream classes derive.
    StreamException A base class for stream exceptions.
    StreamFileException An exception for File errors.
    TArrayStream Parameterized subclass that wraps an array-like buffer with a stream interface.
    WriteException Thrown when unable to write data to Stream.
